oxide colorscheme for [iTerm2](https://iterm2.com/).

## Installation

1. Download `oxide.itermcolors` from this repository
2. Open iTerm2 → Settings (⌘+,) → Profiles → Colors
3. Click **Color Presets** → **Import**
4. Select the downloaded file
5. Click **Color Presets** again and choose **oxide**
